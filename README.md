# Welcome to my resume page!

Here you will find the source code for my resume website, built with React and Chakra UI.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- **React** - UI library
- **Chakra UI** - Component library
- **Vite** - Build tool
- **React Icons** - Icon library

The website features a responsive design with a sidebar navigation and smooth scrolling between sections.
